# Data Science Projects

This repository contains multiple data science and machine learning projects implemented in Python.
Each project focuses on solving a real-world problem using data analysis and modeling techniques.

## Projects Included

### 1. Uber Data Analysis
Analysis of Uber ride data to uncover patterns in demand, trip frequency, and operational insights.

### 2. Travel Insurance Analysis
Exploratory data analysis and prediction related to travel insurance claims.

### 3. RFM Analysis
Customer segmentation using Recency, Frequency, and Monetary (RFM) analysis.

### 4. IPL Analysis
Data analysis on IPL cricket data to extract performance and trend insights.

### 5. Google Sharing Analysis
Analysis related to data sharing or collaboration patterns.

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Scikit-learn

## Note
These projects are being restructured into **individual repositories**, each with detailed documentation and setup instructions.
